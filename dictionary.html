$connection->real_escape_string($variable)
Takes into account the current character set of a MySQL connection and must be used with a mysqli connection object.
get_magic_quotes_gpc()
Checks if the magic_quotes state for GPC (Get/Post/Cookie) operations is enabled. If it is enabled when all quotes and NUL's are escaped with a backslash automatically
stripslashes($variable)
Strips a string of it's slashes. (used in conjunction with get_magic_quotes_gpc()
htmlentities($variable)
Removes HTML from a string
strip_tags($variable)
Strips a string of any html tags. Should be used before htmlentities()
